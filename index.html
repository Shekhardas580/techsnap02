Frontend Web Developer Development Application

  • Chennai
  • Tamilnadu
  • Internship

Job Description


Our team is looking for a Front-end Dev intern with experience on frontend interfaces. You will help define the system architecture, own and implement specific components, and help shape the overall experience. You must be comfortable working as part of a team while taking the initiative to take lead on new innovations and projects. The ideal candidate will be responsible for designing, developing, testing and debugging responsive web and mobile applications for the company and also work in coordination with cross-functional teams to design, develop, and maintain our next generation websites and web tools. Using JavaScript, HTML and CSS, this candidate will be able to translate user and business needs into functional frontend design.


Responsibilities


Build reusable code and libraries for future use


Designing, developing and testing Ul for mobile and web applications


Accurately translate user and business needs into functional frontend code


Continuously maintaining the template engine


Maintain and improve website and also Optimize applications for maximum speed


Requirements


Bachelor's degree or equivalent in Computer Science


Any under graduate can apply


Proven work experience as a Front-end developer [ Personal Projects considered)


Proficient understanding of web markup languages, including HTML5, C553


Familiarity with code versioning tools such as Git ute,


Perks


Internship Certificate with Global ink


Letter of Recommendation (Based on work)


Flexible Timings
